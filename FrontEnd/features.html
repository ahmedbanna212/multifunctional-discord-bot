Bota's Features

1. Add Role

Bota is able to add roles to members of your discord server without having to do all the manual/leg work by yourself, Bota does it for you!

Method: --addrole <@member>

2. Avatar

Bota is able show you avatars of members of your discord server.

Method:
--Avatar <@member> role

3. Remove Role

Bota is able to remove roles from members of your discord server.

Method:
--removerole <@member> role


4. Ban Member

Bota is able to ban members out of your discord server.

Method: --ban <@member>

5. Unban Member

Bota is able to unban members that you have preivously banned from your server.

Method: --unban <@member>

6. Kick Member

Bota is able to kick members out of your server.

Method: --kick <@member>


7. MEMES!!

Bota is able to share a meme from a subreddit to your discord server!.

Method: --meme

8. Leveling System

Bota implements a leveling system to your discord server which keeps track of the level of all members.

Leveing Method: every message that a user sends is counted.

9. Riot Profile

Bota is able to fetch user data from op.gg which contains Riot Games user data and displays it.

Method: --Riot riot ID


10. Leaderboard

Bota is able to show the top 5 members based on their level.

Method: --leaderboard

11. Discord Activities

Bota enables you to play discord Activities like: Youtube, Poker, Chess, Checkers, Wordsnack, among others.

Activities Method:--start activity name

12. Music

Bota offers you a full music player service containing commands such as play, queue, pause, resume and disconnect.


13. Welcome

Bota welcomes all newcomers and members to your server.

14. Translate

Bota is able to translate text from one language to another.

Translate Method:--translate languageacronyum text

15. Help

Bota displays all available features and the method on how to use them.

Help Method: --help
